Cómo Funciona

tree_lib captura un fotograma de la cámara del sistema, extrae entropía de los datos de píxeles centrales y la utiliza para generar una cadena numérica pseudoaleatoria de 1024 dígitos (0–9).

Todo en memoria, sin acceso a archivos ni hilos. Ideal para dispositivos embebidos y aplicaciones seguras.

Resumen de la API

tree_lib::Generator
generate() → Captura un fotograma de la cámara y devuelve un número de 1024 dígitos.
getLast() → Devuelve el último valor generado.

Ejemplo: 

#include "tree_lib.hpp"
#include <iostream>
int main() {
  tree_lib::Generator g;
  std::cout << g.generate() << std::endl;
}

Compilación Manual

Para compilar tree_lib en Ubuntu: 

g++ -c tree_lib.cpp -o tree_lib.o `pkg-config --cflags opencv4`
g++ main.cpp tree_lib.o -o mi_programa `pkg-config --cflags --libs opencv4`
./mi_programa
Requisitos: Ubuntu + GCC + OpenCV 4.
