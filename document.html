Documentación Completa de la Biblioteca tree_lib

Introducción

La biblioteca tree_lib es una herramienta potente y eficiente desarrollada en C++ para la gestión y manipulación de estructuras de datos de tipo árbol. Ofrece funciones para creación, modificación, recorrido y eliminación de árboles.

Características Principales

  • Estructura flexible para representar árboles generales.
  • Recorridos eficientes: preorden, postorden e inorden.
  • Operaciones básicas de inserción, eliminación y búsqueda.
  • Implementación sencilla y ligera, adecuada para proyectos académicos y profesionales.

Instalación

Incluye los archivos tree_lib.hpp y tree_lib.cpp en tu proyecto.

#include "tree_lib.hpp"

Clases y Métodos

Clase TreeNode

Atributos:

  • int value: valor del nodo.
  • std::vector<TreeNode*> children: punteros a nodos hijos.

Métodos:

  • TreeNode(int val): constructor.
  • void addChild(TreeNode* child): añade un hijo.

Clase Tree

Atributos:

  • TreeNode* root: nodo raíz.

Métodos principales:

  • Tree(): constructor.
  • void setRoot(TreeNode* node): establece raíz.
  • TreeNode* getRoot(): retorna la raíz.
  • void insert(int parentValue, int newValue): inserta un nodo.
  • TreeNode* find(int value): busca un nodo.
  • void remove(int value): elimina un nodo.

Ejemplo de Uso

#include "tree_lib.hpp"
#include <iostream>

int main() {
    Tree myTree;
    TreeNode* root = new TreeNode(10);
    myTree.setRoot(root);

    myTree.insert(10, 20);
    myTree.insert(10, 30);
    myTree.insert(20, 40);

    TreeNode* foundNode = myTree.find(30);
    if (foundNode) {
        std::cout << "Nodo encontrado con valor: " << foundNode->value << std::endl;
    }

    myTree.remove(20);
    return 0;
}

Métodos de Recorrido

  • Preorden: Nodo raíz, hijos de izquierda a derecha.
  • Inorden: Hijo izquierdo, raíz, hijo derecho (en árboles binarios).
  • Postorden: Hijos de izquierda a derecha, luego la raíz.

Consideraciones de Rendimiento

Optimizado para operaciones básicas. Precaución con árboles profundos por el uso de memoria.
